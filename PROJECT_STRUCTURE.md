---

## What is NOT committed to Git

| Path / Pattern | Reason |
|---|---|
| `runs/` | YOLOv8 training output — weights, logs, plots (large files) |
| `*.pt`, `*.pth` | Model weights — download from Kaggle or release |
| `data/`, `datasets/` | Raw LLVIP images — download from official source |
| `llvip_fusion_yolov8/` | Auto-generated fused images and YOLO labels |
| `offline_metrics/` | Auto-generated evaluation plots |
| `.ipynb_checkpoints/` | Jupyter autosave artifacts |
| `__pycache__/` | Python bytecode |
